Additional Project

Landed Cost Modeling & Value Engineering

Built a full landed-cost model for the propulsion system and used it to identify practical cost-down opportunities without requiring new tooling.

Overview

This work created visibility into true system cost and turned that visibility into cross-functional decision-making. The model was used to prioritize realistic savings opportunities tied to design and sourcing choices.

What I Built

  • First full landed-cost model for the propulsion system
  • Tradeoff framework for value-engineering opportunities
  • Decision support for cross-functional cost-down work
