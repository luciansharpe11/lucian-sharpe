Additional Project

Smart Pillbox Product Development

Designed and prototyped a smart pillbox integrating enclosure design, PCB electronics, and an embedded microcontroller system.

Overview

This project combined enclosure design, PCB electronics, and embedded control into a compact smart pillbox prototype. The work also included programming the ATmega in C++ and designing a PCB that supported Wi-Fi data upload.

What I Built

  • Enclosure and device architecture for a connected smart pillbox
  • ATmega-based embedded system programmed in C++
  • PCB design supporting Wi-Fi data upload and iterative simplification for manufacturability
