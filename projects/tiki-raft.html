Fun Project: Tiki Raft

Built a fun small-watercraft project that blended practical fabrication, buoyancy thinking, and a bias toward making something real and usable.

Overview

This project was lighter in tone but still rooted in real engineering decisions: structure, flotation, stability, and what it would take to make the concept work reliably on the water.

What I Built

  • Concept and layout work for a small raft-style build
  • Early structure and flotation thinking around practical materials
  • A playful fabrication project with real hands-on problem solving
